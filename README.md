# How to get started?

During development you have to switch to the `develop` branch!

### Start the Nuxt.js Application

``` bash
# install dependencies
$ npm install

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```
